## Naming Convention

### Markdown

```text
verb-topic.md
```

Contoh:

```text
recover-grub.md
install-docker.md
```

### Scripts

```text
verb-purpose.sh
```

Contoh:

```text
backup-home.sh
mount-nas.sh
```
